### 学习方法总结
#### 先感性认识（如有某种需求、因而想要实现某种功能），需要的技术现学。简单来说就是，先找到切实的目标
##### 反例：大学上来学了一堆理论，实操一个都不会（就是我，快把书和知识点吃了也还是不会编程）。不止是编程，其他实用技能的学习亦如此。
#### 原理比编程重要，首先要逻辑清晰
#### 常用的方法、类库一定要滚瓜烂熟：信手拈来才行
